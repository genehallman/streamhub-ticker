Module: TickerView

TickerView

new require("TickerView")(opts)

TickerView is a view of Streamhub data that is structured as a horizontal ticker, similar to television media's ticker, time based, LTR. When the feedCollection option is specified, the ticker view will use the content items in the feedCollection to populate the Y-axis of the ticker, stacking all feedCollection items above the appropriate Ticker items, based on the create date of the content.
